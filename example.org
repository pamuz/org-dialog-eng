#+TITLE: org-ai-prompt Examples

* Getting Started

This file demonstrates how to use org-ai-prompt to send prompts to Claude
from within your org files.

** How to Use

1. Position your cursor anywhere inside a #+BEGIN_PROMPT block
2. Press C-c C-c
3. Claude's response appears in a #+BEGIN_RESPONSE block below

* Tips

** Keep Prompts Focused

Each PROMPT block is independent - Claude only sees that specific prompt, not
your whole document. This keeps responses focused and tokens low.

** Multiple Prompts

You can have as many PROMPT blocks as you want in a single file. Each one is
independent.

* Troubleshooting

If C-c C-c doesn't work:

1. Check if org-ai-prompt-mode is enabled (look for " AI" in mode line)
2. If not, run: M-x org-ai-prompt-mode
3. Make sure Claude CLI is installed: claude -p "test"

See README.md for full troubleshooting guide.
